Send SMS messages to CRM Leads

Adds ability to send SMS to CRM Leads/opportunities.

This module adds a new menu item Send an SMS to Action menu and other features.

This module extends the functionality of our other modules:

  • TurboSMS client
  • AlphaSMS client

How to use this module

CONFIGURATION

Configuration is not needed.

USAGE

To use this module, you need to:

  • Go to CRM - Opportunities menu.
  • Select one or multiple lead.
  • Go to Action and will found some wizards to send sms.
