Sending SMS via TurboSMS service

Allows send SMS messages to partners.

Installation instructions in Russian: Installing and configuring the TurboSMS client.

How to use this module

Configuration

To configure this module, you need to:

  • Configure Keychain key.
  • Configure your provider account.

USAGE

To use this module, you need to:

  • Go to partner menu.
  • Select one or multiple partner.
  • Go to more and will found some wizards to send sms.
